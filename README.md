# Lightweight Python LLM Chatbot

This project is a **lightweight Python chatbot** that communicates with a local LLM (Large Language Model) through REST API requests.  
It is designed to be **modular, reliable, and easy to extend**.

## Features
- Structured JSON requests for communication
- Timeout handling and error management
- Reusable `SimpleChat` class for clean architecture
- Easy to run locally with minimal setup
